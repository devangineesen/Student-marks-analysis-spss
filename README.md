Project Overview

This project focuses on predicting student marks/performance using IBM SPSS Modeler by analyzing different academic and behavioral factors that influence student outcomes. The aim is to apply statistical analysis and predictive modeling techniques to identify patterns in student data and estimate academic performance.
Using IBM SPSS, the project performs data preprocessing, analysis, and prediction to better understand how different variables contribute to student marks. This project was developed as part of an IBM academic subject to gain practical exposure to predictive analytics and data-driven decision-making.
________________________________________
❓ Problem Statement

Educational institutions often face challenges in identifying students who may struggle academically. Traditional evaluation methods only reflect performance after examinations, making it difficult to provide early intervention.
The problem addressed in this project is:
How can student academic marks be predicted using available student-related data to identify performance trends and support academic improvement?
Predicting marks in advance can help educators and institutions make informed decisions and support students effectively.

⚙️ Methodology

The project follows a structured predictive analytics workflow.
1. Data Collection
Student academic data was collected and organized into a structured format suitable for analysis.
2. Data Preprocessing
The dataset was cleaned and prepared by:
•	Removing missing values 
•	Handling inconsistent records 
•	Formatting data properly 
•	Selecting relevant attributes 
3. Data Analysis
Exploratory analysis was performed to understand relationships between variables affecting student performance.
4. Model Building in IBM SPSS
A predictive model was created using IBM SPSS Modeler Stream to analyze patterns and estimate student marks.
The process included:
•	Importing dataset 
•	Data transformation 
•	Variable selection 
•	Statistical analysis 
•	Model training 
•	Prediction generation 
5. Result Evaluation
The predicted student performance was evaluated to determine the effectiveness of the model.

📈 Results

The predictive model successfully analyzed student-related factors and generated estimated academic performance outcomes.
Key Findings
•	Attendance has a noticeable impact on marks. 
•	Study habits influence student performance. 
•	Previous academic records contribute significantly to prediction accuracy. 
•	Multiple academic indicators collectively improve prediction quality. 
The model demonstrated how educational data can be used to support performance analysis and prediction.

✅ Conclusion

This project demonstrated the practical implementation of predictive analytics using IBM SPSS Modeler for student marks prediction.
By analyzing educational data, the system helps identify performance trends and estimate student marks more effectively. Such predictive systems can support educators in understanding student progress and enable proactive academic assistance.
Overall, the project highlights the importance of data-driven educational analysis in modern learning environments.
